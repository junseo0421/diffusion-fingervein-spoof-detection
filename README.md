# Synthetic Finger Vein Image Generation for Spoof Detection Using Diffusion Models  
DDPM 기반 finger-vein 합성 이미지 생성으로 스푸핑 (spoofing) 공격을 모사하고, 생성 이미지를 fake 샘플로 활용해 real/fake 안티-스푸핑 분류기 (ConvNeXt-small)를 학습하는 미니 프로젝트입니다.

---

## 0. 프로젝트 요약

- **목표**: 실제 스푸핑 데이터가 부족한 환경에서, **확산 모델 (DDPM)** 로 합성 정맥 이미지를 생성하여 **안티-스푸핑 분류기** 학습에 활용 가능함을 확인
- **데이터**: Open finger-vein dataset인 HKPU-DB 기반 DDPM 학습 후 fake 이미지 생성
- **분류기**: ImageNet 사전학습 **ConvNeXt-small** fine-tuning (마지막 head 위주)
- **결과**: **Accuracy 81.07%**

## 4. DDPM 학습 및 이미지 생성
### 4.1 학습 (Train)
ddpm/main.py는 absl flags 기반입니다. 주요 설정은 다음과 같습니다.

- T=1000, batch_size=2, lr=2e-4
- 로그/체크포인트 저장: --logdir (기본 ./logs/HKPU_1)


실행 예시:
```bash
python ddpm/main.py \
  --train \
  --logdir ./logs/HKPU_1 \
  --img_size 64 \
  --batch_size 2 \
  --lr 2e-4 \
  --T 1000 \
  --total_steps 800000
```

학습 중
- logdir/sample/ 아래에 샘플 이미지가 주기적으로 저장됩니다.
- ckpt_last.pt, ckpt_best.pt가 저장됩니다.

### 4.2 학습된 DDPM으로 이미지 생성 (샘플링)
- --eval 모드는 logdir/ckpt_last.pt를 로드해 생성 이미지를 저장합니다.

```bash
python ddpm/main.py \
  --eval \
  --logdir ./logs/HKPU_1 \
  --img_size 64 \
  --num_images 50000
```

생성 결과는 기본적으로 아래 경로에 저장됩니다.

```bash
logs/<Dataset_name>/generated_images_last/
└── 000000.png
└── 000001.png
└── ...
```

- 생성 결과 중 완전한 노이즈 or 붕괴 사례는 제외하고, 정맥 구조가 유지된 샘플을 fake 데이터로 사용했습니다.

---

## 5. Real/Fake 분류기 학습 (classifier)
classifier/train.py는 real 이미지 + ddpm 생성 fake 이미지를 합쳐서 학습합니다.
Dataset은 glob (real_root/*/*) 형태로 real 이미지를 수집하고, fake는 fake_root/*.png를 수집합니다.

---

## 6. 결과 요약
- ConvNeXt-small 학습은 약 70 epoch 이후 안정적으로 수렴 경향 
- 최종적으로 별도 real/fake 검증 세트에서 Accuracy 81.07% 
- diffusion 기반 합성 데이터가 단순 증강보다 더 실제 분포에 가까운 스푸핑 샘플로 활용될 수 있음을 논의

## 7. Future Work
